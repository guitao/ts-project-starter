# typescript + webpack + mocha + chai 工作流模板 



:relaxed: 开发：自动编译

`npm start`

http://localhost:8080/webpack-dev-server/bundle

:relaxed: Mocha测试：用`typescript`写测试

`npm test`


:relaxed: 发布：发布到build目录下，压缩代码

`webpack`
